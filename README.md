# Supermarket-Grocery-Sales---Retail-Analytics

## Project Overview

This project analyzes supermarket retail sales data to identify sales trends, product performance, and key business insights using Python.

## Objectives

* Clean and preprocess retail sales data
* Perform exploratory data analysis (EDA)
* Identify top-performing categories and products
* Analyze sales and profit trends
* Generate actionable business insights

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook

## Key Analysis

* Sales Trend Analysis
* Category-wise Performance
* Product-wise Performance
* Profitability Analysis
* Business Insights
